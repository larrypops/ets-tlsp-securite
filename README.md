# ETS TLSP Sécurité Privée

Site vitrine Next.js App Router pour ETS TLSP Sécurité Privée à Douala.

## Développement

```bash
npm install
npm run dev
```

## Déploiement Vercel

Utiliser le preset **Next.js** avec la racine du dépôt comme Root Directory.

Commande de build:

```bash
npm run build
```

Les images du site sont servies depuis `public/images`.
